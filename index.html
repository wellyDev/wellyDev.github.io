Sobre Nós

A WellyDev é uma companhia nova que tem origem devido as dificuldades que micro e pequenas Empresas encontram ao procurar por desenvolvedores com preços acessíveis para sistemas simples, nosso compromisso é disponibilizar condições para qualquer pessoa física/juridica poder criar seu website ou sistema proprio.

SISGO é um sistema desenvolvido pela WellyDev Com a intenção de torna mais acessível as tecnologias para pequenos escritórios de odontologia. O SISGO é totalmente web feito em PHP JavaScript e boostrap Com funções recursiva e otimizado para sistemas mobile, acesse o SISGO Do seu Navegador, tablet e celular.

Web Developer: Trabalhamos com Desenvolvimento web Back-End e Front-End. Nossa equipe trabalha com linguagens variadas como Java(Spring), PHP, C#(asp.net), JavaScript e Ruby On Rails, etc...

Desktop Developer: No Desenvolvimento Desktop trabalhamos com C#, Python, JavaScript(electron) e Java, criamos sistemas para modernizar o máximo possível sua empresa.

Mobile Developer: No Desenvolvimento Mobile nos esforçamos o máximo possível para adaptar seus sistemas para qualquer dispostivo tais quais Android e IOS. trabalhando com linguagens como Java, dart(flutter) e JavaScript.
